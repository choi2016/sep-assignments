Calculating tip at restaurant:

Input bill and numPeople
If numPeople >= 6
	Tip = Bill * 0.20
else 
	Tip = Bill * 0.15
end

Deciding number of layers to wear:

Input tempOfTheDay in Fahrenheit
Let numLayers = 0
if tempOfTheDay >= 70
	numLayers = 1
elsif tempOfTheDay < 70 and tempOfTheDay > 40
	numLayers = 2
elsif tempOfTheDay < 39
	numLayers = 4
else
	numLayers
end

Deciding whether to ride bus or ride share:

Input distance in miles
if distance > 2
	call rideShare
else
	call bus
end
